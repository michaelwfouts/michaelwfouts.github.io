Project Portfolio

The table below contains all of Michael's Projects, searchable by topics and relevant technologies. Click on the Project Title to be taken to the GitHub Repository.


Project Title Description Technologies
End to End YouTube Comments Dashboard This project collects and analyzes YouTube comments, results being displayed in a dashboard to monitor the overall sentiment of a YouTube channel and the amount of engagement a channel gets. Python (pandas, plotly, dash, nltk), SQL, Google BigQuery, YouTube API, Google App Engine
Bayesian Analysis of Substrate Binding Models This project uses a Markov Chain Monte Carlo technique to assess three proposed models of substrate binding of WISP1 to Collagen I in the context of Breast Cancer Metastasis. Project from Graduate Statistics for Biomedical Engineers (BMEG 601) at WVU. R Programming Language (ggplot2, Bolstad2), MCMC (Metropolis-Hastings, Gelman-Rubin), Biology (Model Proposals)
Analysis and Modeling of Stock Fundamentals This project collects fundamental data for stocks in the S&P 500, performs an analysis on the metrics, and creates a supervised regression and classification ML model as well as a k-means unsupervised clustering. Python (pandas, beautiful soup, XGBoost, scikit-learn, seaborn), Yahoo! Finance API, Web Scraping, Machine Learning
Global Energy Analysis This project was developed to identify high quality data sources about global power generation and analyze it to help others easily understand differences in countries' energy portfolios. Python (choropleth maps, plotly), Energy (Power Generation, Biomass Production), Geolocation (FIPS)
Music and Lyric Analysis This project analyzes the musical content (based on Spotify ML Features) and lyrics of punk pop act Stand Atlantic to glean insights on how I can write and produce better music of my own. Python (pandas), R Programming Language (ggplot2, tidytext, dplyr), NRC Sentiment/Emotion Analysis, Spotify API, Genius API
Financial Advice Document Analysis This project completes an NLP analysis to glean insights from financial advice PDFs found on the internet and serve as a framework to quickly analyze any corpus. Python (k-means clustering, TF-IDF, Word Clouds), Machine Learning, NLP, PDF parsing
Interactive Retirement Savings Dashboard This interactive visualization allows users to investigate their retirement scenarios based on a few of their current financial metrics. R Programming Language (Shiny, plotly)
Monthly Spending Analysis This project provides an R Markdown Analysis for how I analyze my finances every month and sample data so that others can analyze their spending habits as well. R Programming Language (plotly, modelr), Regression
Kaggle 2021 Survey Dashboard This dashboard shows an analysis I conducted on the Kaggle 2021 Survey of their users to glean a better understanding of the Data Science Field. Tableau
Advanced Reactions (CHE 625) This repository shows selected work associated with the Graduate Level Advanced Reactions (CHE 625) Course at WVU taken in the Spring of 2023, focusing on reactor design. Python (scipy, seaborn, pandas), Chemical Engineering, Reactor Design, Aspen Plus
